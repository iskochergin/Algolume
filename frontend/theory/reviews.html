Отзыв от Марины Панковой (учитель информатики)

Рецензия на проект Кочергина Ивана «Algolume: Пошаговый визуализатор алгоритмов олимпиадного программирования»

В своём проекте Иван рассмотрел одни из самых популярных алгоритмов олимпиадного программирования. Они требуют не только понимания технической реализации кода, но и, самое главное, визуального представления работы алгоритма. Для каждого из рассмотренных алгоритмов Ваня подобрал оптимальные формы визуализации, которые наглядно отражают механизм работы алгоритма. Реализованные в проекте кнопки навигации позволяют «путешествовать» по алгоритму, что способствует лучшему восприятию кода программы.

Кроме визуализации работы алгоритма в проекте Ивана также представлена и визуализация кода, а именно состояние стека переменных: перечислены все переменные, задействованные в коде, и описаны их состояния на каждом шаге выполнения программы. Таким образом процесс интерпретации программного кода делается прозрачным.

Особенно хочется отметить интеграцию различных технологий по созданию визуализации. Для визуализации графа Иван использует дополнительный модуль pyvis, позволяющий создавать интерактивные визуализации графов. Данные наблюдения позволяют сделать выводы о большой работе, проделанной Иваном по исследованию интерфейсов визуализации и выбору оптимальных решений.

Проект Ивана позволит проводить занятия по изучению описанных в проекте алгоритмов с большей наглядностью. Кроме того, возможность исследования своего кода является важным шагом не только в освоении конкретного алгоритма, но и способствует развитию алгоритмического мышления в целом. Я планирую использовать проект Ивана на занятиях по Олимпиадной подготовке - 2, так как данные алгоритмы как раз входят в программу изучения.

Замечания по улучшению проекта:

По работе с интерфейсами алгоритмов:

  1. При запуске алгоритма пользователю становится недоступной информация по условию задачи, в том числе формату входных данных, что может привести к некорректному вводу данных.
  2. Не обнаружен интуитивно понятный способ изменения данных в поле input. Пока найден способ только через обновление страницы.
  3. При вводе данных после ввода данных последней строки ввода необходимо нажать Enter, иначе данные последней строки либо не будут считываться (как на странице алгоритма DFS), либо стираются (как на странице алгоритма Дейкстры).

По работе сайта:

  1. Ссылка на Feedback открывается в том же окне, логичнее сделать в новой вкладке, так как это сделано для вкладки GitHub.
  2. На главной странице рекомендую сделать более подробное описание проекта, в том числе описать интерфейс работы с программой, что именно скрывается под «визуализацией» алгоритмов.

Рекомендации по развитию проекта:

  1. Рекомендую добавить визуализацию алгоритма бинарного поиска по ответу. На примере одной из классических задач, например «Провода».
  2. Рекомендую добавить визуализацию алгоритма BFS – поиска в ширину.
  3. Будет полезным добавить для каждого алгоритма ссылки на сайты с проверяющей системой, где можно порешать задачи на эти алгоритмы. Например, выбрать сайт с большим набором учебных задач https://informatics.msk.ru/.

Отзыв от Владимира Гуровица (учитель информатики)

Представленный проект предназначен для визуализации алгоритмов с целью помочь школьникам освоить эту непростую область знаний. Написанный для этого сайт обладает достаточным функционалом и приятным, привлекательным дизайном. UX в целом понятный, хотя есть отдельные предложения по его усовершенствованию. К сожалению, пока реализовано лишь несколько алгоритмов для демонстрации возможностей сайта. Для использования проекта на практике потребуются еще значительные усилия по наполнению сайта контентом.

Проект выполнен на высоком техническом уровне, проделана большая работа по имплементации библиотеки для выполнения кода и для удобного отображения интерфейса на мобильных устройствах.

Проект представляет собой веб-сайт с визуализацией популярных алгоритмов (ДП, DFS, Дейкстра), который позволяет пользователям не только изучить теоретические основы алгоритмов, но и наблюдать за их пошаговым исполнением. Отзыв от Никиты Токалова (олимпиадный программист)

Ресурс Algolume.ru предоставляет материалы для изучения различных алгоритмов из олимпиадного программирования. В первую очередь хочется отметить приятный интерфейс сайта и его отзывчивость к изменению размеров устройства, то есть контент остаётся читабельным и на телефоне, и на планшете, и на ноутбуке. Также хочется отметить то, что ресурс обладает уникальной среди подобных себе функцией: визуализировать алгоритмы с любыми входными данными. Это большой плюс в их изучении, потому что самому расписывать, например, таблицу для динамического программирования для лучшего понимания может быть трудозатратно, а ресурс предоставляет такую возможность. Также ресурс прекрасно подходит для отладки собственных программ, потому что предоставляет возможность визуализировать свой код (!). Все визуализации также реализованы аккуратно и красиво, таблицы и графы хорошо читаемы.

Из недостатков можно отметить то, что некоторые формулировки или объяснения могут быть непонятны новичкам в изучении алгоритмов, и с текущими текстами объяснений ресурс, скорее, больше подойдёт для тех, кто уже имеет представление об алгоритмах, но хочет получить более глубокое понимание.

Я думаю, что если продолжать развивать ресурс: добавлять различные алгоритмы и делать их визуализации, то Algolume может стать незаменимым для многих ресурсом для изучения олимпиадной информатики!
